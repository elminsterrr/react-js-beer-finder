# react-js-beer-finder


## Getting Started

### Installation

Open a Terminal and run:

```sh
cd react-js-beer-finder
```

and run:

```sh
npm install
```

### Usage

Start the app by typing:

```sh
npm start
```

### Browser

In your browser go to:

```sh
http://localhost:3000/
```

### Testing

Start testing by typing:

```sh
npm run test
```
